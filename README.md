# Solar Miner ☀️⛏️

A fun browser-based clicker game built with **HTML, CSS, and JavaScript** for a school project.

In **Solar Miner**, players mine crystals from asteroids, purchase upgrades, automate mining systems, and explore deep space to become the ultimate galactic mining company.

---

# Features ✨

- Click asteroids to collect crystals
- Buy powerful mining upgrades
- Passive income system
- Unlock new planets and sectors
- Animated UI effects
- Save system using Local Storage
- Responsive design for desktop and mobile

---

# Gameplay 🎮

## Basic Mechanics

- Click the asteroid to earn crystals
- Spend crystals on upgrades
- Upgrades increase mining speed and passive income

## Upgrades

Examples:
- Mining Drill
- Auto Miner
- Laser Extractor
- Mining Drone
- Quantum Harvester

## Progression

As players earn more crystals, they unlock:
- New planets
- Better mining technology
- Faster automation systems
